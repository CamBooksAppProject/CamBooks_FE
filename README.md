# CamBooks: 대학생을 위한 통합 도서 플랫폼

## 제작 동기

대학생들은 매 학기 새로운 전공 서적을 구매해야 하고, 사용한 교재를 다시 판매하기도 합니다. 그러나 기존의 중고거래 플랫폼은 도서 거래에 최적화되어 있지 않으며, 학습 커뮤니티와 소통 기능은 여러 서비스에 흩어져 있어 불편함이 존재합니다.

특히 같은 캠퍼스, 같은 과목 수강생들 간의 **신뢰 기반 도서 거래**와 **학습 정보 공유 공간**이 부족한 점에 착안하여,  **거래 + 커뮤니티 + 소통**을 하나의 서비스로 통합한 **CamBooks**를 기획하게 되었습니다.

## 작품 설명

**CamBooks**는 대학생이 도서 관련 모든 활동을 하나의 앱으로 해결할 수 있는 **올인원 플랫폼**입니다.
대학생의 캠퍼스 라이프를 보다 경제적이고 지식 친화적으로 바꿀 필수 서비스가 되는 것을 목표로 합니다.

### 핵심 가치
- **편의성**
  - ISBN 검색 및 바코드 스캔 기반 도서 등록
  - 중고거래, 커뮤니티, 채팅을 단일 앱에서 제공  
- **신뢰성**
  - 대학 이메일 인증으로 안전한 거래 환경 보장  
- **커뮤니티**
  - 같은 수업/전공 기반 스터디, 정보 공유 커뮤니티 제공  

## 주요 기능

- **중고 도서 거래**
   - ISBN 검색 기반 정확한 도서 정보 등록
   - 캠퍼스 기반 필터링으로 근거리 거래 가능
- **학습 커뮤니티**
   - 과목별 스터디 / 소모임 생성 및 참여
   - 일정 관리 및 인원 설정 기능 제공
- **실시간 채팅**
   - WebSocket 기반 1:1 / 그룹 채팅 지원
   - 거래 및 커뮤니티 내 즉각 소통 가능
- **통합 검색 & 스크랩**
   - 도서·커뮤니티·게시글을 한 곳에서 탐색
   - 관심 정보 스크랩 기능으로 재확인 편리


## 기술 스택

| 영역 | 기술 |
|-----|-----|
| **Frontend** | React Native, Expo Go |
| **Backend** | Spring Boot, Spring Security, JWT, JPA, MySQL |
| **실시간 기능** | WebSocket, Redis |
| **외부 API** | 네이버 도서 검색 API |

## 기대효과 및 차별점

- **대학생 + 도서**에 완전히 특화된 플랫폼
- 대학 인증 기반 **신뢰도 높은 거래 환경**
- 교재 구매 부담 완화 및 중고 거래 활성화
- 스터디 / 학습 정보 공유로 **캠퍼스 지식 네트워크 형성**
